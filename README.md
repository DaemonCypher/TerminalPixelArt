# TerminalPixelArt
ASCII character art generator

# Running the code on Docker
```
docker build -t art .
docker run -it --rm art
```
# Running the Code Locally
```
python3 main.py
```

# Example of art Drawn on CLI 
```
Type the file path to the image file: example.png
Do you want to invert the image (Y)es/(N)o: y
How wide (number of pixels) would you like the image to be: 50
```
